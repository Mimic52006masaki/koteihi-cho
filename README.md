# koteihi-cho

## 概要

このアプリケーションは、個人の固定費を効率的に管理するためのツールです。
月々の固定費の予定額と実績額を記録し、支払日や支払口座と紐付けて管理することができます。
また、複数の口座を登録し、それらの残高をダッシュボードで一元的に把握することで、家計全体の資産状況を可視化します。

### 主な機能

- **固定費管理**: 月ごとの固定費の予定と実績を記録し、支払状況を追跡します。
- **支払い履歴**: 各固定費の支払い情報を口座と紐付けて記録し、詳細な履歴を確認できます。
- **口座管理**: 複数の銀行口座や支払い口座を登録し、それぞれの残高を管理します。
- **ダッシュボード**: 全口座の合計残高や月々の固定費の合計額など、家計の主要な財務状況を一覧で確認できます。

### 技術スタック

- **バックエンド**: PHP
- **フロントエンド**: React (TypeScript)
- **データベース**: MySQL (または互換性のあるRDBMS)
